# Madrid Interactive Map

An interactive map of Madrid including metro lines, tourist places, monuments, museums, parks, activities, and shopping locations.

---

## 📍 Features
- Interactive Leaflet map
- Metro network with dynamic line display
- Nearby places search (1000m radius)
- Geolocation support
- Tourist points of interest categorized
